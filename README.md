# Sai Sumanth Reddy Kachi - Portfolio

A modern, fully responsive portfolio website showcasing my experience as a Full-Stack Engineer with expertise in MERN applications and AI integration.

## Features

- 🎨 Modern design with smooth animations
- 📱 Fully responsive across all devices
- ✨ Interactive UI with scroll animations
- 🚀 Fast performance with optimized assets
- 🔗 Direct contact links and social integration

## Sections

- **Hero** - Eye-catching introduction with typewriter effect
- **About** - Professional background and statistics
- **Experience** - Detailed work history with achievements
- **Skills** - Tech stack including frameworks, databases, cloud, and AI/ML
- **Projects** - Featured projects with metrics
- **Education** - Academic credentials
- **Contact** - Multiple ways to get in touch

## Tech Stack

- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts

## Local Development

To run locally:

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Or use http-server directly
npx http-server -p 3000
```

Then open `http://localhost:3000` in your browser.
